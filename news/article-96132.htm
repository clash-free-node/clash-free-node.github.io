四、v2ray毒药的配置教程：从入门到精通

1. 修改 config.json 配置文件（适用于手动用户）

v2ray毒药的主要配置文件为 config.json，路径通常在程序根目录。

json
{
  "inbounds": [{
    "port": 1080,
    "protocol": "socks",
    "settings": { "auth": "noauth" }
  }],
  "outbounds": [{
    "protocol": "vmess",
    "settings": {
      "vnext": [{
        "address": "example.com",
        "port": 443,
        "users": [{
          "id": "UUID-替换成你的UUID",
          "alterId": 0
        }]
      }]
    }
  }]
}

你可以根据自己的节点服务商提供的信息，修改 "address"、"port"、"id" 等字段。

2. 配置协议与传输方式

v2ray毒药支持以下主流传输方式：

  • TCP + TLS（最稳定）

  • WebSocket + TLS（伪装性强，适合通过443端口绕墙）

  • mKCP + 混淆参数（速度快但容易被识别）

  • gRPC（新兴协议，性能高但对服务端要求较高）

3. 设置分流规则（路由配置）

通过配置 "routing" 字段，实现按域名或 IP 路由规则分流。

例如：

json
"routing": {
  "rules": [{
    "type": "field",
    "domain": ["geosite:cn"],
    "outboundTag": "direct"
  }, {
    "type": "field",
    "domain": ["geosite:google", "geosite:facebook"],
    "outboundTag": "proxy"
  }]
}
